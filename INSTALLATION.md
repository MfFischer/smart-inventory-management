# BMSgo Smart Inventory System - Installation Guide

## System Requirements

- Windows 10 or later (64-bit)
- At least 4GB of RAM
- At least 500MB of free disk space
- Internet connection (for initial setup)

## Installation Instructions

1. **Download the BMSgo executable**
   - Extract the ZIP file to a location of your choice

2. **Run the application**
   - Double-click on `BMSgo.exe` to start the application
   - A command window will appear, followed by your default web browser opening to the application
   - **Important:** Do not close the command window while using the application

3. **First-time setup**
   - On first run, you'll need to create an admin account
   - Follow the on-screen instructions to set up your business profile

4. **Using the application**
   - The application runs locally on your computer
   - All data is stored locally in a database file
   - You can access the application at any time by running `BMSgo.exe`

## Troubleshooting

If you encounter any issues:

1. **Application won't start**
   - Make sure you have extracted all files from the ZIP
   - Try running as administrator (right-click > Run as administrator)

2. **Browser doesn't open automatically**
   - Open your browser manually and go to: http://127.0.0.1:5000

3. **Database errors**
   - If you see database errors, the database file may be corrupted
   - Contact support for assistance with database recovery

## Uninstallation

To uninstall BMSgo:
1. Delete the folder containing the BMSgo executable and associated files
2. No registry entries or system files are modified by this application
